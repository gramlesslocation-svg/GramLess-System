# 🌾 Gramness ERP System - Corporate Prototype

This repository houses the end-to-end rapid cloud prototype for the **Gramness Rice Co.** Enterprise Resource Planning (ERP) tracking ecosystem. 

## 🏗️ Core Business Engine Specifications
- **Live Inventory Balance Matrices:** Manages supply line dependencies across all **four proprietary rice variants** (`Rice Team`, `Rice Gold`, `Rice White`, and `Rice Creamy`) inside specific packaging metrics (`800g` / `2.4kg`).
- **Logistics & CEO Fleet Management Map:** Visualizes live shipping locations via geographical coordinates.
- **Dynamic Ledger Billing System:** Generates live structural receipts and updates operational data tables instantly.

## 🚀 How to Launch Locally
1. Clone the project code down to your workstation:
   ```bash
   git clone https://github.com
   ```
2. Run installation configurations:
   ```bash
   pip install -r requirements.txt
   ```
3. Boot the local server node:
   ```bash
   streamlit run app.py
   ```
